# Code of Conduct

Проект придерживается принципов Contributor Covenant (адаптировано).

## Наши обязательства
Создавать доброжелательную, уважительную и безопасную среду для всех участников, независимо от опыта,
пола, идентичности, национальности, вероисповедания и прочих признаков.

## Ожидаемое поведение
Уважение, конструктивная критика, эмпатия, готовность помочь новичкам.

## Недопустимо
Оскорбления, харассмент, дискриминация, публикация чужих личных данных, троллинг.

## Обеспечение
Мейнтейнеры вправе удалять/отклонять вклад, нарушающий кодекс. Сообщать о нарушениях — через контакты
из `SECURITY.md`.
