# HOME_REWORK_V2 — живой остров Логова (v36)

## 1. Результаты анализа

Логово в v35 — это меню с узкой канвас-полосой 180px (`mountLivingLair` в 05-ui-base.js): эмодзи-драконы с лёгким ИИ на 9 состояний, тап открывает свиток. Камеры нет, окружения нет, яйца живут в отдельном экране-меню, остров не меняется с прогрессом. При этом фундамент оказался отличным: характеры (`NATURES`) и настроение (`happy`) уже влияют на поведение, спрайты и стадии роста есть, rAF-цикл корректно останавливается вне экрана, сохранение (`persist`) терпимо к новым полям объектов. Ключевая архитектурная находка: `renderLair()` вызывает `mountLivingLair(wrap)` — модуль, загруженный после 05-ui-base.js, переопределяет эту функцию, и вся переработка выполняется **без правки существующих JS-файлов**.

## 2. Концепция

Главный герой — дракон, остальное усиливает связь с ним. Логово — парящий остров шириной 1.2–2.6 экрана с камерой, живущий по реальному времени суток и дневной погоде. Остров растёт вместе с игроком: восемь зон открываются уровнями 1→100, драконы «отстраивают» новые территории. Из Hatch Dragons взяты только ощущения — плотность мира, чувство дома, связь с питомцами; интерфейс, композиция, стиль и палитра — собственные (эмодзи-арт и пергаментная стилистика Draconis).

## 3. Реализованные изменения (новый модуль js/05b-island.js, ~520 строк)

**Камера.** При входе — мягкий перелёт к активному дракону (lerp, `CAM_EASE`), он приветствует репликой; выбор другого дракона тапом плавно переводит камеру; свободное перетаскивание пальцем; резких скачков нет.

**Живой остров.** Восемь зон (`ISL_ZONES`): поляна с домом и костром, пруд с кувшинками, водопад с мостиком и ручьём, кристальный сад, руины с тотемом, горячие источники с паром, утёс, святилище. ~70 объектов: деревья, кусты, цветы, грибы, скалы, фонари, тропинка из камней, места отдыха — процедурная раскладка с сидом, всё выглядит частью острова.

**Живое окружение.** Птицы (ночью — летучие мыши), бабочки днём, светлячки ночью, падающие листья, качающаяся на ветру растительность, облака-параллакс, рыбы выпрыгивают из пруда, звёзды с мерцанием, искры света. Всё — из фиксированных пулов.

**Поведение.** 17 состояний: к прежним добавлены «ест» (любимая еда по стихии), «сидит у воды», «греется у костра», «наблюдает за другими», «исследует», «идёт в любимое место», «смотрит на яйца», «строит». Выбор автоматический, взвешенный.

**Личность.** Использованы существующие `NATURES` (боевой баланс не тронут): характер задаёт веса поведения и слово в подписи («смелый», «ленивый», «любознательный»…). Проверено симуляцией: стремительный днём летает и играет, живучий ночью спит и греется.

**Любимые места.** У каждого дракона — своя точка (детерминирована от uid), возвращается туда каждые 25–60 секунд с эмоцией 🏡.

**Зона инкубации.** Гнездо 🪺 в мире: яйца видны с кольцом прогресса цвета стихии, готовые покачиваются. Тап — карточка: происхождение (стихия, глубина биома), прогресс, переход в Гнездо; драконы сбегаются посмотреть.

**Интерактивность.** Тап по любому объекту — пружинное покачивание и искры (кристаллы светятся своим цветом), по дракону — эмоция, реплика, выбор и доверие, по яйцу — карточка. Отклик есть даже без награды.

**Время суток.** Реальные часы → непрерывная фаза: ночь/утро/день/вечер с плавным смешением цветов неба (`islMix`), солнце движется по дуге, ночью — луна, звёзды, светлячки, свет фонарей/кристаллов (shadowBlur), драконы больше спят. Баланс не меняется.

**Погода.** Дневная (сид от даты): солнце, ветер (двойное качание, быстрые листья), дождь (пул капель), туман (градиент), далёкая гроза (редкие вспышки), звёздная ночь. Только атмосфера.

**Доверие.** `d.trust` 0–100 (хранится в объекте дракона — существующее сохранение, миграций не нужно). Растёт: ласка (с кулдауном 20с), мини-истории, реакции у гнезда. Пороги 10/25/50/80/100 открывают: новые реплики (5 наборов `ISL_PHRASES`), эмоцию 💞, записи Кодекса (`worldSeen`), на максимуме дракон приносит украшение из существующей системы `DECORATIONS`. Боевые характеристики не изменяются.

**Социальное поведение.** Каждые 9–16 секунд сцена пары: здороваются 👋, играют, отдыхают рядом, спорят ❕, летают вместе, наблюдают.

**Мини-истории.** Каждые 38–70 секунд у случайного дракона появляется маркер ❗: найден камень, потерялась игрушка, старое яйцо, новая пещера, древний тотем, тайник. Награды — только доверие и записи Кодекса (косметика/коллекция).

**Развитие Логова.** Зоны открываются по `progLevel()` (1/5/10/20/35/50/70/100); закрытые видны как туманные силуэты с подписью «🔒 · ур.X» — игрок видит, куда расти. При новом открытии (`S.islandLvlSeen`) — празднование: баннер «✨ зона открыта!», тост, половина стаи летит «строить» (состояние build 🔨). Ширина мира растёт с числом зон.

**Интерфейс.** Сцена — главный элемент экрана (высота до 50% вьюпорта), подпись — одна строка, карточки событий — лёгкий оверлей поверх мира в существующем стиле `enc-card`. Прежние панели (подарок дня, инкубатор-строка, свиток) сохранены ниже — привычные функции не потеряны.

## 5. Производительность

Object Pool для всех частиц (капли 70, листья 14, светлячки 22, птицы 4, бабочки 6, рыбы 3, искры 40) — ноль аллокаций в кадре; градиент неба кэшируется и пересоздаётся раз в ~3 минуты; горизонтальный куллинг всех объектов и агентов по камере (~27 объектов в кадре из ~70); DPR ≤ 2; один rAF, который сам останавливается при уходе с экрана (существующий паттерн); эмодзи-рендер вместо текстур — переиспользование глифов системного шрифта; агентов ≤ 12. Замеров деградации нет: бюджет кадра сопоставим с прежней полосой (больше объектов, но без теней/фильтров на всём, кроме 2–4 светящихся ночью).

## 6. Критерии приёмки

Логово живое (окружение, погода, время суток, социальные сцены) ✓; первым виден активный дракон (перелёт камеры + приветствие) ✓; драконы взаимодействуют с миром (вода, костёр, гнездо, любимые места, стройка) ✓; остров растёт с прогрессом и это празднуется ✓; ощущение меню убрано (мир — главный элемент, UI минимален) ✓; архитектура сохранена (модуль поверх, ноль правок в старых файлах) ✓; производительность без деградации (пулы, кэши, куллинг) ✓.
