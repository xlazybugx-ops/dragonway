# EGG_SYSTEM_V2.md — яйца как главная коллекция

Яйца превращены в главный предмет коллекционирования: полноценный **данные-каталог** `EGG_CATALOG`
(данные отдельно от логики), 6 редкостей, типы с собственными источниками, гарантия (pity) в конфиге,
инкубация игровыми действиями (без реальных таймеров), **уникальные** и **секретные** яйца, Кодекс Яиц v2,
переработка (пыль + осколки, без продажи). Расширение существующей системы, без слома боя, экономики, UI,
локализации, архитектуры.

Легенда: ✅ — в коде · 🧩 — данные готовы, точечная интеграция расписана.

---

## 1. Новая структура — каталог (данные отдельно от логики) ✅

`EGG_CATALOG` (в `js/02-data-content.js`) — авторитетный список типов яиц. Каждая запись:

`id · name · desc · rarity(1-6) · el · drop(отн. шанс) · source · cond(условие) · look{emoji} · dragons[] · unique · secret · fixed`

Логика (получение/кодекс/переработка) читает **только** из каталога и из конфига `GB.Eggs`.

## 3. Типы яиц ✅ (16 записей)

- **Стихийные (5):** Огненное, Ледяное, Ядовитое, Грозовое, Теневое — базовый источник (бои/исследование).
- **Тематические (5):** Лесное 🌿, Кристальное 💎, Королевское 👑, Древнее 🥚, Первозданное ✴️.
- **Уникальные (3):** ♔ Яйцо Пепельного Короля → Пламевластец; ❄️ Яйцо Северного Хранителя → Аврорин;
  🐍 Яйцо Изумрудного Леса → Мирозмей. Фиксированный дракон, **один раз за игру**.
- **Секретные (3):** 🌘 Яйцо Затмения, 🌩️ Яйцо Грозовой Короны, ⭐ Яйцо Первой Стихии — скрыты до условия.

Тип определяет **пул возможных драконов** (`dragons[]`); редкость задаёт уклон к редким видам этого пула.

## 4. Источники (у каждого типа — свой, без единой таблицы) ✅

`GB.Eggs.sources` задаёт пол редкости и буст для каждого источника; `rollEggFromSource(src, tier)` катит
яйцо именно из пула этого источника.

| Источник | Пол редкости | Типы оттуда |
|---|---|---|
| `battle` обычные бои | 1 | стихийные |
| `elite` элитные враги | 2 | стихийные (выше) |
| `boss` боссы | 4 | Древнее ✅ (подключено к победе над владыкой) |
| `tower` башня | 5 | Первозданное |
| `daily` задания | 2 | необычные |
| `streak` серии побед | 3 | Королевское |
| `explore` исследование | 1 | Лесное |
| `chest` редкие сундуки | 3 | Кристальное |
| `secret` секретные события | 5 | секретные |

✅ Подключено: **boss** → яйцо при победе над владыкой. 🧩 Остальные источники — один вызов
`rollEggFromSource('<src>')`/`addCatalogEgg('<id>')` в соответствующем событии (список в §12).

## 5. Гарантированное получение (pity) ✅

`GB.Eggs`: `pityStep 4 · pityCap 60 · pityDiv 30`. Чем дольше без Эпического+, тем выше шанс редких (до ×3
на band Эпик+), при выпадении — сброс. Порядок редкостей сохраняется. Все значения — в конфиге.

---

## 6. Инкубация — только игрой ✅

Реального ожидания нет. Прогресс идёт за действия: победа в бою (+2), победа в полёте (+2), находка в
полёте (+1). `GB.Eggs.incBattle/incExplore`. incNeed по редкости: 0/3/6/10/16/24. Кнопка «Высидеть»
заблокирована до готовности. 🧩 Доп. источники прогресса (босс/задание/сундук) — один вызов `incubateEggs(n)`.

---

## 7. Уникальные яйца ✅

`unique:true` + `fixed:<вид>`: выдаются через `addCatalogEgg(id)` только **один раз** (`S.eggsUnique`),
дракон фиксирован. Пример: Яйцо Пепельного Короля → гарантированный Пламевластец.

---

## 8. Секретные яйца ✅

`secret:true` + `cond`: скрыты в Кодексе (🔒) до выполнения условия; `unlockSecretEgg(id)` раскрывает и
выдаёт. Пример условий: «босс без единого лечения» (✅ подключено → Яйцо Затмения, трекинг `b.usedHeal`),
«открой все сундуки региона», «бой одной стихией без потерь».

---

## 9. Кодекс Яиц v2 ✅

Раздел в Гнезде рендерится **из каталога**: для каждого яйца — изображение, редкость (цвет рамки),
описание, источник, открыто ли, возможные драконы, сколько раз получено (`S.eggStats`). Секретные — 🔒 до
открытия. Прогресс «открыто N/16 · %».

---

## 10. Ценность и переработка ✅

Яйца **не продаются**. Переработка (`recycleEgg`): пыль `recycleBase + rarity·recyclePerRarity` +
**осколки** 🔮 для редкости 4+ (`rarity·shardsFromRarity`, поле `S.shards`). Осколки — материал для будущих
систем (не валюта продажи).

---

## 11. Баланс вероятностей

Базовое яйцо (источник `battle`, ярус I): Обычное ~61% · Необычное ~25% · Редкое ~10% · Эпическое ~3% ·
Легендарное ~1.5% · Древнее ~0. Глубже (ярус III) и у сильных источников пол редкости выше: `boss` — всегда
Эпическое+, `tower` — Легендарное+. Pity гарантирует, что редкое **достижимо** без зависимости от чистой
удачи. Редкие яйца заслуживаются источником (босс/башня/серия/секрет), а не только выпадают.

## 13. Рекомендации по интеграции источников (готово к внедрению)

1. **elite/streak/daily/chest/tower/explore**: добавить `rollEggFromSource('<src>')` в события —
   элитная победа, серия побед (счётчик), выдача награды задания, открытие редкого сундука, зачистка башни,
   находка в полёте.
2. **Секреты**: «все сундуки региона» → счётчик открытых сундуков на регион → `unlockSecretEgg('egg_stormcrown')`;
   «одна стихия без потерь» → флаг состава пачки + `usedHeal` → `unlockSecretEgg('egg_purity')`.
3. **Уникальные**: привязать к сюжетным вехам (первый босс каждого мира → своё уникальное яйцо).
4. **Осколки**: сделать сток (обмен на гарантированное яйцо нужной редкости) — «жалость-магазин».
5. **Анимация вылупления по редкости**: усиливать эффекты ритм-мини-игры с ростом редкости.
