# Security Policy

## Область
Клиентское веб-приложение без бэкенда. Данные игрока хранятся локально (`localStorage`), не передаются
на сервер. Сетевых секретов в репозитории нет.

## Сообщить об уязвимости
Открой приватный issue или свяжись с мейнтейнером репозитория. Просьба не публиковать детали до
исправления.

## Рекомендации
- Не добавлять внешние скрипты/трекеры без ревью.
- Проверять сторонние ассеты/шрифты (сейчас только Google Fonts).
- `localStorage` не хранит чувствительных данных.
