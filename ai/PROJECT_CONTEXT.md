# PROJECT_CONTEXT.md — постоянный контекст Draconis

> Прочти этот файл первым. Он даёт любому разработчику или ИИ полное понимание проекта.

## Что это
**Draconis** — браузерная одностраничная RPG-коллекционер драконов для iPhone Safari. Приручение
драконов, инкубация яиц, пошаговый и аркадный бой, исследование-полёт, живое логово, кузница,
скрещивание, шпиль талантов, кодекс, боссы, ежедневки. Жанр-настроение: **уютная сказочная книга**.

## Технологический стек
Чистый ванильный стек, **без фреймворков и сборки**: HTML + одна CSS + 13 JS-модулей. Отрисовка —
DOM + `<canvas>` + `requestAnimationFrame`. Звук — WebAudio-синтез. Хранение — `localStorage`.
Деплой — статический (GitHub Pages, `.nojekyll`), файлы отдаются из корня репозитория.

## Архитектура (критично понимать)
- **13 модулей в общем ГЛОБАЛЬНОМ скоупе**, подключаются `<script>`-тегами в `index.html` в СТРОГОМ
  порядке (00→11). Дубли верхнеуровневых `const/let` → SyntaxError. Нет модулей ES/бандлера.
- Порядок: `00-balance` (конфиг GB) → `01-data-core` → `02-data-content` → `03-state` →
  `04-systems` → `05-ui-base` → `06-world` → `07-flight` → `08-battle` → `08b-arcade` →
  `09-forge-breed` → `10-spire-codex` → `11-save-init`.
- **`GB` / `GAME_BALANCE`** (00-balance) — единый конфиг всех коэффициентов баланса. Меняя игру,
  правят GB, а не код.
- Состояние игры — объект `S` (03-state), сериализуется в localStorage (`draconis_save_v1`).
- Кэш-бастинг: у всех css/js тегов `?v=N` (текущая N в index.html). При изменении ассетов N растят.
- Ассеты — `images/` с fallback (procedural/onerror), поэтому замена ассетов безопасна.

## Структура репозитория
`index.html` (корень) · `js/` · `css/` · `images/` · `design/` (дизайн-система+токены) · `prompts/`
(AI-промпты) · `ai/` (этот контекст и правила) · `docs/` (аудиты/отчёты/история) · `tests/` · `tools/`
· `_archive/` (устаревшее). Подробно — `PROJECT_STRUCTURE.md`.

## Художественное направление
**Cozy Storybook Fantasy** — тёплая детская книжка: мягкие формы, ручная роспись, светлая палитра,
лавандовые тени, дружелюбные драконы. Правила — `design/`. Токены — `design/tokens/`.

## Незыблемые правила
Не менять баланс (GB) без осознанного решения; не ломать порядок загрузки; не переименовывать
глобальные символы бездумно; сохранять обратную совместимость сейвов; держать деплой-файлы в корне.
