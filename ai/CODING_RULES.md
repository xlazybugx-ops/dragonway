# CODING_RULES.md


- Ванильный JS, ES2019-совместимо (Safari iOS). Без ES-модулей/сборки.
- Крупные файлы править точечно (python/heredoc), не целиком.
- Игровые циклы — в try/catch, dt-clamp, самоостановка вне видимой области.
- DOM-ссылки в кадре кэшировать; слушатели снимать при остановке (AbortController/removeEventListener).
- Новые поля `S` — с дефолтом и `||`-guard; обратная совместимость сейвов обязательна.
- Магические числа баланса — только в `GB`. Без DEBUG/console.log в проде (warn в catch — можно).
