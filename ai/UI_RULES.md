# UI_RULES.md


- Использовать только токены `design/tokens/` (цвета/радиусы/тени/типографика/отступы).
- Единый стиль кнопок/карточек/панелей (см. design/BUTTON_GUIDE, CARD_GUIDE, PANEL_GUIDE).
- Нижняя навигация, bottom sheet, крупные скругления, воздух, мягкие тени, ≥44px цели.
- Минимум текста; второстепенное — под спойлер/лист. Одна главная задача на экран.
- Уважать `prefers-reduced-motion` и режимы доступности.
