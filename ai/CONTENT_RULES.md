# CONTENT_RULES.md


- Новый контент (дракон/яйцо/биом/босс) добавляется как строка в декларативную таблицу — без правки логики.
- Драконы — персонажи, не монстры; яйца — читаемая редкость; биомы — своя палитра-настроение.
- Имена ассетов — строго по design/ASSET_NAMING.md (совпадение с ключами кода).
- Тексты — русский, единый тёплый тон.
