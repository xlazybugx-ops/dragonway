# ARCHITECTURE_RULES.md


- 13 модулей, общий глобальный скоуп, строгий порядок загрузки (00→11). Не нарушать порядок.
- Никаких дублей верхнеуровневых `const/let/function`-имён между модулями.
- Данные — в декларативных таблицах (SPECIES, EGG_CATALOG, WORLD_BOSSES, MILESTONES…) и `GB`.
- Логика читает данные через геттеры; представление не хранит правил баланса.
- Никаких внешних зависимостей/фреймворков/бандлеров без веской причины.
- Проверка: `node --check` каждого файла + конкатенации в порядке загрузки.
