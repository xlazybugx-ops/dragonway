# PROMPT_RULES.md


- Каждый промпт: неизменяемая STYLE-преамбула + блок категории + NEGATIVE (prompts/AI_PROMPT_LIBRARY.md).
- Палитра/материалы/свет — из токенов и MASTER_STYLE_PROMPT. Один сид-диапазон на серию.
- Прозрачность и размеры — по шаблону; итоговый файл — по габаритам заменяемого ассета.
- Брак → перегенерация с той же преамбулой. Референсы утверждённого — для image-to-image.
