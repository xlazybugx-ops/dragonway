# ECONOMY_REPORT.md — экономика и прогрессия (сводка)

Полная сводка источников и стоков ресурсов после аудита. Значения — из `GB` (00-balance); баланс не
изменялся. Подробная диагностика темпа — в PROGRESSION_AUDIT.md, поток игрока — в PLAYER_FLOW.md.

---

## 2. Кривые

- **Опыт:** `45·lvl^1.15` — плавно, ~1 ур./8–12 мин.
- **Золото/бой:** `lvl·(8+rarity·4)·risk·legend`.
- **Кузница:** `15·(1+rarity·8)·1.5^lvl`. **Кузня:** `600·1.85^lvl`.
- **Логово:** 800 / 2 600 / 6 500 / 15 000 🪙 (+пыль, +боссы на ур.4–5).
- **Инкубация:** `incNeed [0,3,6,10,16,24]`, +2/бой, +1/находка.

## 4. Баланс источник/сток

Ранняя игра: щедрый приток + частые открытия удерживают вовлечённость. Средняя (ур.8–60): приток
стабилен, но открытий систем нет — компенсируется квестами/вехами/коллекцией/логовом и, главное,
**подсказкой следующего шага**, которая всегда даёт понятную цель. Экспоненциальные стоки (кузня/
кузница) поглощают поздние излишки.

## 5. Изменения этой итерации

Механику экономики не трогали. Добавлена **система «Следующий шаг»**: одна ближайшая цель с наградой
и разблокировкой; второстепенные данные — под спойлер. Устраняет риск «что делать?» в среднем этапе.

## 6. Рекомендации

1. Ввести мини-механику-открытие около ур.20–30 (заполнить средний провал).
2. Второй сток для ⭐звёзд владык (сейчас только Восхождение).
3. 🔮осколки как ускоритель/скидка в кузнице (второй сток).
4. Настройка кривой золота логова по метрикам, если ур.4–5 ощущаются далёкими.
