# TECH_DEBT.md — технический долг (независимый)

По коду билда v33. Факты, без предположений.

## Архитектура
- **13 JS-модулей в общем глобальном скоупе**, строгий порядок загрузки через `<script>`. Дубли
  верхнеуровневых `const/let` → SyntaxError. Валидность подтверждена (`node --check` конкатенации
  чист), но модель хрупкая: нет модулей ES/бандлера, всё держится на дисциплине. **Средний долг.**
- Нет системы сборки (нет минификации/бандла/tree-shaking). Отдаётся как есть.
- Кэш-бастинг ручной (`?v=33` правится вручную во всех тегах).

## Размер файлов/функций
- Крупнейшие модули: `02-data-content` 824, `06-world` 802, `08-battle` 796, `07-flight` 714 строк.
- Крупнейшие функции: `frameBody` (полёт, ~226 стр — игровой цикл), `endBattle` (~118), `renderDetail`
  (~122), `renderMap`/`openMarket` (~70). Большие, но для игрового кода терпимо. **Низкий долг.**

## Дублирование
- Два рандом-хелпера: `rnd` (04, целочисленный) и `rand` (08b, дробный) — разные, но путают.
- `dist`/`clampN` локально в 08b дублируют общую логику. **Низкий долг.**
- Сцены (логово/полёт/аркада) повторяют паттерн rAF+dt-clamp+try/catch — можно вынести в общий движок.

## Магические числа
- В основном вынесены в `GB` (00-balance) — сильная сторона. Остаточные литералы в рендер-коде
  (размеры/цвета/тайминги) присутствуют, но некритичны.

## Мёртвый/отладочный код
- **DEBUG `console.log`** в `07-flight.js:278` (`'полёт v12: экран собран…'`) — убрать. **P3.**
- `console.warn` в catch-блоках (07/08b) — легитимная обработка ошибок, оставить.
- Функции `selectDragon`/`lairSorted`/старый путь карусели стали неиспользуемыми после реворка логова —
  определены, но не вызываются. Мусор. **P3.**

## Локализация
- **i18n отсутствует.** Весь UI-текст захардкожен на русском в JS/HTML. Смены языка нет. Для RU-аудитории
  беты — не блокер; для масштабирования — долг. **P2.**

## Конфигурация/данные
- Данные — в JS-объектах (`GB`, `SPECIES`, `EGG_CATALOG`, `WORLD_BOSSES`, `MILESTONES`…). Внешних
  JSON/конфиг-файлов нет. Отделение данных от логики частичное (в коде, но декларативное).

## Репозиторий/гигиена
- В корне лежат **прототипы** (`arcade-prototype.html` 25 КБ, `den-prototype.html` 31 КБ) — не
  подключены в `index.html`. **Убрать из релизной сборки. P2.**
- 40+ `.md`-отчётов и `DEN-COMBAT-REPORT.pdf` (84 КБ) в корне — раздувают репозиторий. **P3.**

## Тесты
- **Автотестов нет.** Только `node --check` (синтаксис). Ни unit, ни e2e, ни визуальных. **P1** для
  качества, хотя не блокирует closed beta.

## Итог
Долг умеренный и в основном «гигиенический» (глобальный скоуп, отсутствие бандла/тестов, артефакты в
репо). Критических архитектурных дефектов, ломающих игру, не обнаружено.
