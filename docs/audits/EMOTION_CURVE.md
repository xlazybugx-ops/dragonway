# EMOTION_CURVE.md — эмоциональная кривая игрока

Реконструкция по коду (события, награды, празднования, гейты). Отмечены пики и провалы.

## Кривая по этапам

| Момент | Эмоция | Чем вызвано (код) |
|---|---|---|
| Онбординг, имя дракона/поселения | 🙂 вовлечение | `11-save-init` онбординг |
| Первое яйцо | 🎉 радость/предвкушение | церемония `eggCeremony` |
| Первое вылупление | ✨ открытие | звук `hatch`, новый вид в кодексе |
| Первая победа (гарантирована) | 😌 уверенность | `tutorialGuard`, звук `win` |
| Первый левел-ап | ⬆️ гордость | `levelUpFx` вспышка+вибро |
| Кузница@3 / Шпиль@5 / Гнездилище@8 | 🤩 открытие | `FEATURE_MIN` дрип |
| Стрик/подарок дня | 🎁 привычка/удовлетворение | `STREAK_REWARDS` (день 7 легендарный) |
| **Ур.8 → ~60** | 😐→😴 **скука/усталость** | **нет открытий систем — «мёртвый» участок** |
| Открытие биома II (ур.60) | 🗺️ облегчение/новизна | `BIOME_MIN_LEVEL` |
| Первый босс (ур.100) | 😮 напряжение/триумф | FSM-фазы, уникальное яйцо + легенда |
| Редкое яйцо/морф | 😍 удивление | наследование окраса, пити |
| Заполнение кодекса | 🏆 гордость коллекционера | вехи, % знаний |

## Провалы без эмоций
- **Ур.8–60 (главный):** длинный участок без новых систем; эмоции держатся только на инкрементальных
  наградах и коллекции. Риск «плато». Смягчено подсказкой «Следующий шаг», но пиков открытий нет.
- Повторяющиеся арена-бои среднего этапа — эмоционально ровные.

## Пики (хорошо расставлены)
Ранняя игра насыщена микро-пиками (яйцо/вылупление/победа/уровень/открытие механики). Поздняя — боссы
и коллекция. Провал — строго в середине.

## Рекомендация
Вставить эмоциональный пик-открытие около ур.20–30 (новая мини-механика/событие/мини-босс), чтобы
разорвать плато ур.8–60.
