# DRAGON_REWORK.md — уникальность драконов

Каждый дракон теперь ощущается своим: боевая **роль**, **уникальная пассивка** (мех. отличие), данные
коллекции (обитание/еда/факт), ролевой **ИИ врагов** и точечный ребаланс. Расширение существующей системы
(SPECIES/натуры/древо талантов/морфы) — без слома боя, экономики, сюжета, арт-стиля, архитектуры.

Легенда: ✅ — в коде · 🧩 — спроектировано.

---

## 1. Боевые роли ✅

Все 15 видов распределены по архетипам (данные в `DRAGON_META`, `js/02-data-content.js`):

| Роль | Виды |
|---|---|
| Защитник | Магмарок, Гляциар, Вечнолёд |
| Воин | Эмберлинг, Пламевластец |
| Берсерк | Жарёныш, Умбракс, Бездномор |
| Стрелок | Темпестар, Громозов |
| Следопыт | Спорокрыл, Сумрачник |
| Контроллер | Гнилоклык, Мирозмей |
| Поддержка | Аврорин |

Роль определяет пассивку, уклон характеристик и поведение при использовании врагом.

## 2. Уникальные пассивные способности ✅

Каждый вид — **своя пассивка** (все 15 уникальны, проверено). Работают через боевые хуки
(`critPct/vampPct/healPct/manaRegen` + новые `dmgRedPct/dodgePct`). Применяются к дракону игрока;
на баланс влияют умеренно (5–15%).

| Вид | Пассивка | Эффект |
|---|---|---|
| Эмберлинг | Тёплая кровь | +1 реген маны |
| Жарёныш | Хвост-факел | +10% крит |
| Магмарок | Лавовое нутро | −12% урона |
| Пламевластец | Венец пламени | +6% крит, +3% лечение |
| Гляциар | Ледяной панцирь | −10% урона |
| Вечнолёд | Несокрушимый | −15% урона |
| Аврорин | Северное сияние | +4% лечение, +1 мана |
| Спорокрыл | Споровый шлейф | +8% вампиризм |
| Гнилоклык | Гнилое дыхание | +5% вамп, −4% урона |
| Мирозмей | Кольца мира | −8% урона, +3% лечение |
| Темпестар | Седлок ветров | 10% уклонение |
| Громозов | Скорость молнии | +8% крит, 5% уклонение |
| Умбракс | Тенеход | +12% крит |
| Сумрачник | Бесшумный | 8% уклонение, +4% вамп |
| Бездномор | Эхо пустоты | +10% вамп, +5% крит |

Новые боевые хуки (`js/08-battle.js`, `doStrike`): **уклонение** (шанс свести урон к 0) и
**снижение урона** защитника. Оба ограничены и безопасны.

## 3. Базовые характеристики — проверка и ребаланс ✅

Проанализированы все 15 видов. Устранены «всё-в-одном» легендарные (были сильны по всем статам):

| Вид | Было | Стало | Причина |
|---|---|---|---|
| Бездномор | def 13 · spd 11 | def **10** · spd **13** | стеклянный ассасин: слабее в защите, но быстрее (концепция сохранена) |
| Мирозмей | atk 21 | atk **18** | контролёр-танк, а не топ-урон |

Остальные виды уже имеют выраженные сильные/слабые стороны (телосложения: колосс/вихрь/страж). Ни один вид
не доминирует по всем параметрам.

---

## 4. ИИ врагов по роли ✅

Когда вид используется врагом (`foeTurn`), поведение соответствует роли: **Защитник** чаще встаёт в защиту
(×1.7), **Маг/Стрелок** чаще применяют сильные приёмы (×1.4–1.5), **Берсерк/Воин** чаще бьют мощным ударом
(0.7/0.55). Пассивки к врагам не применяются — их идентичность выражается поведением (баланс врагов не тронут).

---

## 5. Коллекционная ценность ✅

В Кодексе у каждого открытого вида показаны: **роль**, **пассивка**, а в подсказке — **обитание**,
**любимая еда** (по стихии) и **интересный факт** (данные в `DRAGON_META` + `FAVFOOD`). Редкость, стихия,
собранные окрасы — как прежде.

---

## 6. Активные способности — анализ 🧩

Сейчас приёмы (`MOVES`) заданы по стихии (2 на стихию) и общи для видов одной стихии; уникальные заклинания
даёт древо талантов (`SPELL_POOLS`, по видам). Дубликатов «почти одинаковых» заклинаний нет — базовый и
сильный приём различаются силой/риском, а талант-спеллы тематичны по стихии. **Уникальность видов внутри
одной стихии обеспечивают пассивки и роль** (мех. отличие). 🧩 Рекомендация: со временем дать 2–3 видам с
одинаковой стихией разные talent-финишеры (эксклюзивные ульты) для ещё большей дифференциации.

## 7. Спроектировано (готово к внедрению) 🧩

- **Прогрессия пассивки**: усиливать пассивку на вехах уровня (напр. −12% урона → −15% на ур.50) через
  множитель от `d.level` в `dragonPassiveFx`.
- **Специализация** (после уровня X): выбор ветви (напр. Огненный → «Разрушитель» +крит/урон или
  «Хранитель» +защита/лечение). Реализуется как поле `d.spec` + модификатор к пассивке; внешний вид не меняется.
  Опирается на существующие развилки древа талантов (A/B-узлы).
- **Бонус за разнообразие команды**: в составе для арены/волн давать бонус за разные стихии/роли и редкие
  комбинации — поле-множитель к награде/статам вне боя (без влияния на 1-в-1 расчёт).
- **Визуальная идентичность**: цвет дыхания/следа/глаз/когтей по виду — данные `DRAGON_META.tint`; применяется
  в SVG-силуэте и следе полёта. Без нагрузки на FPS.

## 9. Рекомендации по дальнейшему балансу

1. Плейтест пассивок: если суммарный «power creep» игрока заметен — снизить верхние значения (−15% → −12%).
2. Ввести специализацию (§7) как главный слой выбора «стиля игры» на 25–50 уровнях.
3. Бонус за разнообразие команды (§7) — стимул собирать разные роли/стихии, а не фармить один топ-вид.
4. Визуальная идентичность (§7) — дешёвый, но сильный шаг к «каждый дракон свой».
