# LAIR_REWORK.md — логово как дом стаи

Цель: логово перестаёт быть меню и становится домом, куда хочется возвращаться. Живая стая на
экране, минимум панелей, одно понятное следующее действие.

---

## 1. Анализ прежнего логова

| Вопрос | Ответ (до) |
|---|---|
| Зачем игрок открывает логово | как хаб: покормить/погладить, открыть свиток, уйти в бой/кузницу |
| Сколько времени проводит | мало — транзитный экран между действиями |
| Что повторяется чаще всего | выбрать дракона → покормить/погладить → уйти |
| Что перегружало экран | панель сортировки, карусель всех карточек, блок Заповедника, деталь — всё сразу |

Вывод: экран был «списком-меню». Живая сцена уже существовала (`mountLivingLair`), но тонула под
управляющими панелями и показывала **всех** драконов, а не активную стаю.

---

## 2. Что стало

**Логово — дом стаи.** Главный экран показывает только четыре вещи:

1. **Стая** — живая сцена (канвас) с драконами активной стаи.
2. **Уровень логова** — компактная полоса `🏰 ур.N · A/Cap 🐉` + `⚙️ Стая` + `⬆️`.
3. **Инкубатор** — компактная полоса ближайшего яйца (прогресс или «готово к вылуплению»).
4. **Одно следующее действие** — крупная кнопка-подсказка по состоянию.

Убрано с главного экрана: панель сортировки, карусель карточек, блок Заповедника. Управление
составом целиком ушло в бот-шит `⚙️ Стая` (стая ⇄ Заповедник, фильтры). Свиток дракона открывается
**тапом по дракону прямо в сцене**.

---

## 3. Поведение стаи (живой дом)

Движок сцены (`_lairPick`/`_lairUpdate`/`_lairDraw`) даёт драконам автономную жизнь; веса поведения
зависят от природы и настроения дракона:

| Поведение | Как проявляется |
|---|---|
| 🌙 Сон | дремлет, покачивание слабое, приглушён |
| 🚶 Прогулки | бродит к случайной точке |
| 🎮 Игры | сходится с соседями, нотки ♪ |
| 😌 Отдых | стоит, чистится (groom) |
| 💬 Общение | сближается с другими драконами |
| 🕊️ Полёты | взлетает выше, быстрее машет |
| 👀 Реакция на игрока | тап → смотрит на тебя, сердечко, открывается свиток |
| ✨ Мини-события | иногда находит цветок/кристалл/бабочку |

Скорость выбора поведения: `up`-ген природы (spd→больше полётов/прогулок, hp→больше сна, atk→рёв/
игры, def→чистка/покой), счастье (радостный→игры/полёты, сонный→сон). Активная стая = только
`!reserve`, до 12 агентов ради Safari.

---

## 4. Одно следующее действие (приоритет)

`lairNextAction()` возвращает единственную подсказку:

1. 🎁 Подарок дня готов → «Забрать подарок».
2. 🐣 Есть дозревшее яйцо → «Высиди готовое яйцо» (→ Гнездо).
3. ⬆️ Хватает на улучшение логова → «Улучши логово».
4. 🥚 Есть яйца → «Загляни в Гнездо».
5. 🗺️ Иначе → «Отправься в странствие».

## 6. Рекомендации

1. Кормление/поглаживание вынести жестами прямо в сцену (тап-удержание) — меньше заходов в свиток.
2. Украшения логова влияют на настроение/поведение стаи.
3. Сон/бодрствование по времени суток устройства.
4. Реакция стаи на возвращение игрока (приветствие после отсутствия).
