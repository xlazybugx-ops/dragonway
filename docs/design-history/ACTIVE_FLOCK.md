# ACTIVE_FLOCK.md — Активная стая

Логово — дом активной стаи. На главном экране живут и ведут себя только драконы стаи; остальные —
в Заповеднике (см. SANCTUARY.md). Состав меняется за один экран.

---

## 1. Что показывает логово

Только активная стая — как живые обитатели дома (канвас-сцена, поведение: сон/прогулки/игры/полёты/
общение/реакция на игрока). Резервные драконы в сцене не появляются. До 12 агентов ради Safari.

Вместимость стаи = уровень логова (`LAIR_LEVELS`: 4→6→8→10→12). Новый вылупившийся при полной стае
уходит в Заповедник (🆕), не пропадая.

---

## 2. Управление (бот-шит ⚙️ Стая)

Кнопка `⚙️ Стая` в полосе уровня открывает нижнюю панель `openFlockManager`:

```
🏰 Активная стая · A/Cap
[🐉][🐉][ ＋ ]            ← тап по дракону → в Заповедник
🌿 Заповедник · N
[Все][🔥][🧊][🟢][⚡][🌑] [⭐][🆕][↕]   ← фильтры
[🐉][🐉]…                ← тап → в стаю
```

Замена в **одном** экране: тап по слоту стаи убирает дракона в Заповедник; тап по ячейке
Заповедника ставит в стаю (если есть место). Никаких переходов между экранами.

---

## 3. Доступ к дракону

Свиток дракона (кормёжка, поглаживание, реликвии, таланты, скрещивание, арена) открывается **тапом
по дракону прямо в живой сцене** — карусель карточек с главного экрана убрана, экран чистый.

---

## 4. Правила

- В стае всегда ≥ 1 дракон.
- Стая ≤ вместимости логова; при полной — сначала освободи слот.
- `toggleReserve(uid)` — единая точка перемещения (логово ⇄ Заповедник).

## 6. Рекомендации

1. Пресеты стай («боевая»/«исследовательская») в один тап.
2. Сила стаи и покрытие стихий в шапке бот-шита.
3. Перестановка порядка стаи перетаскиванием.
