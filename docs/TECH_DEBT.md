# TECH_DEBT.md — технический долг (сводно)

| Долг | Уровень | Заметка |
|---|---|---|
| Нет бандла/минификации; глобальный скоуп | средний | держится на дисциплине; `node --check` конкатенации обязателен |
| Нет автотестов (только node --check) | средний | добавить smoke/unit (tests/) |
| Локализация отсутствует (хардкод RU) | средний | ввести L10N-словарь |

| Дубли `rnd`/`rand`; мёртвые функции после реворка логова | низкий | безопасно вычистить позже |
| Крупные функции (игровые циклы) | низкий | терпимо для game-loop |
| Провал темпа ур.8–60 | P2 (геймдизайн) | мини-механика-открытие |
