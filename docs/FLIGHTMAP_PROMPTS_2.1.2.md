# Flight map prompt set — 2.1.2

> Top-down playable flight map background for an iPhone dragon game. Premium hand-painted fantasy game map, painterly but readable at mobile scale, cohesive Draconis style. Strict 90-degree orthographic bird's-eye view, terrain fills the frame, evenly distributed landmarks, edge-to-edge scrolling map texture. Absolutely no horizon, no sky, no perspective landscape, no characters, no dragons, no labels, no UI, no text, no watermark.
