# Маркеры локаций поселения

Комплект из 12 интерактивных маркеров для карты поселения. Все изображения подготовлены в едином визуальном коде Draconis 3.0: стилизованная фэнтезийная архитектура, тёплый камень, бирюзовая патина, золотые акценты и читаемый центральный символ.

## Технические параметры

- Формат: WebP с прозрачностью.
- Размер: 192 × 192 px.
- Без текста, рамки и встроенной тени интерфейса.
- Безопасный визуальный размер в интерфейсе: 64–112 px.
- Для выбранного состояния рекомендуется внешний золотой ореол, для доступного действия — мягкая пульсация бирюзового свечения.
- Исходники и контрольная таблица находятся в `tmp/imagegen/hub-markers/`.
