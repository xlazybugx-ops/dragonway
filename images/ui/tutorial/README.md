# Велла и элементы обучения

Комплект подготовлен по разделу 5.3 графического аудита.

## Портреты Веллы — 512×512

| Файл | Состояние |
|---|---|
| `tutorial_vella_welcome.webp` | приветствие и знакомство |
| `tutorial_vella_think.webp` | объяснение механики |
| `tutorial_vella_discovery.webp` | новая находка |
| `tutorial_vella_warning.webp` | безопасное предупреждение |
| `tutorial_vella_proud.webp` | успех и освоение механики |
| `tutorial_vella_support.webp` | поддержка после ошибки |

Во всех портретах сохраняются форма рогов, цвет глаз, рисунок чешуи и медальон Кодекса.

## Обучающие накладки

| Файл | Размер | Назначение |
|---|---:|---|
| `tutorial_gesture_tap.webp` | 512×512 | показать одиночное нажатие |
| `tutorial_gesture_drag.webp` | 1024×512 | показать перетаскивание или свайп |
| `tutorial_spotlight.webp` | 512×512 | выделить целевой элемент |
| `tutorial_arrow.webp` | 1024×512 | указать направление или объект |

Все изображения сохранены как WebP lossless с альфа-каналом. Внешнюю прозрачность накладок следует дополнительно задавать CSS, чтобы один файл работал на тёмных и светлых сценах.
