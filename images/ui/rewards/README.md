# Наградные эмблемы

Комплект из 12 эмблем прогресса и наград, восстановленный из последней незавершённой генерации 22 июля 2026 года.

| Файл | Назначение |
|---|---|
| `reward_daily.webp` | ежедневный подарок |
| `reward_streak.webp` | серия входов |
| `reward_streak_protected.webp` | защищённая серия |
| `reward_expedition.webp` | недельная экспедиция |
| `reward_quest_complete.webp` | выполненное задание |
| `reward_discovery.webp` | новое открытие |
| `reward_level_up.webp` | повышение уровня |
| `reward_rare_drop.webp` | редкая добыча |
| `reward_ascension.webp` | восхождение дракона |
| `reward_biome_unlock.webp` | открытый биом |
| `reward_mastery.webp` | мастерство |
| `reward_mystery.webp` | таинственная награда |

## Технические параметры

- Формат: прозрачный lossless WebP.
- Размер: 128 × 128 px.
- Рекомендуемый размер показа: 24–64 px.
- Порядок исходного листа: 4 столбца × 3 строки, слева направо и сверху вниз в порядке таблицы.
- Мастер, keyed-версия, сценарий обработки и контрольный лист: `tmp/imagegen/rewards/`.

## Базовый промт

> Create a strict 4 by 3 contact sheet containing exactly twelve separate reward and progression emblems for the Draconis 3.0 browser game. Cozy hand-painted storybook fantasy, polished gouache, aged gold, turquoise magic, warm chocolate outline, identical visual weight, readable at 48 px, flat chroma-key background, no readable text, letters, numbers, UI panels, watermark or cast shadow.
