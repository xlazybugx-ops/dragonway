# Рамки редкости

Шесть прозрачных рамок единой геометрии для карточек яиц, реликвий и наград.

Порядок прогрессии: `common`, `uncommon`, `rare`, `epic`, `legendary`, `primordial`.

- Формат: lossless WebP с альфа-каналом.
- Размер: 256×256 px.
- Центральное окно и углы прозрачны.
- Ранг меняет материалы и насыщенность декора, но не посадочную геометрию.

Мастер, хромакей-версия, контрольный лист и сценарий нарезки находятся в
`tmp/imagegen/rarity-frames/`.
