# BUILD

Сборка **не требуется** — проект отдаётся как статические файлы.

## Локальный запуск
Нужен статический сервер (из-за политики модулей/кэша Safari лучше не открывать `file://`):
```
# любой из вариантов из корня репозитория
python3 -m http.server 8000
# или
npx serve .
```
Открыть `http://localhost:8000`.

## Проверка кода
```
for f in js/*.js; do node --check "$f"; done
cat js/00-*.js js/01-*.js ... js/11-*.js > /tmp/all.js && node --check /tmp/all.js
```
(конкатенация в порядке загрузки ловит конфликты имён в общем скоупе).

## Кэш
При изменении `css/js/images` поднять `?v=N` во всех тегах `index.html` (обход кэша Safari).
