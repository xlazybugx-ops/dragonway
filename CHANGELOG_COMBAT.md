# CHANGELOG_COMBAT.md — изменения читаемости боя

Баланс, формулы урона, механики боссов и игровой процесс НЕ изменены. Добавлены только визуальные
подсказки, устраняющие пробелы из аудита.

---

## Добавлено (пошаговый бой)

- **Очередь ходов** — полоса `🐲 Ты → 👹 Соперник` с подсветкой активного.
- **Чипы эффектов** под бойцами: защита 🛡️, парирование ⚔️, комбо 🔥×N; у босса — щит 🔰N,
  ярость 😡, застывание 💫 (с подсказками).
- **Маркер цели** ⌖ у соперника.
- **Индикация готовности ульты/заклинания** — золотое пульсирующее свечение кнопки, обновляется
  каждый ход (раньше статус мана-гейта устаревал между ходами).

## Добавлено (аркадный бой)

- **Предпросмотр цепочки** молнии: ломаная перескоков + кольца на целях + счётчик 🎯N до
  подтверждения (дополняет линию/круг/радиус из v28).

## Проверка

`node --check` всех 13 модулей — OK. Конкатенация в порядке загрузки — без конфликтов имён.
Обновление статуса привязано к существующему `renderHpOnly` (обновляется каждый ход) — без новых
циклов и нагрузки на Safari.

## Рекомендации

1. Пошагово визуализировать также точность мини-игры и ожидаемый исход брони.
2. Мини-полоса «инициатива» при будущих боях 2+ участников (сейчас строгая очередь 1×1).
3. Цветовой код чипов эффектов по типу (щит — синий, ярость — красный) для ещё более быстрого чтения.
4. Вынести формы зон (линия/круг/цепочка/конус) в общий реестр для повторного использования.
