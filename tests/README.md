# Тесты тестового релиза

Запуск из корня проекта:

```powershell
powershell -ExecutionPolicy Bypass -File tests/run-tests.ps1
```

Проверяются синтаксис всех игровых модулей, параметры прогрессии, рейтинг боевой силы,
награды арены, генетическая экономика, ограничение пассивного дохода, ссылки на ресурсы
и бюджет изображений релизной сборки (не более 100 МБ, без PNG и резервных копий).
