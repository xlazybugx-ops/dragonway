# 🐉 Draconis · Кодекс Чешуи

Уютная браузерная RPG-коллекционер драконов для iPhone. Приручай драконов, выси­живай яйца, исследуй
сказочные биомы, сражайся, обустраивай живое логово. Настроение — **интерактивная сказочная книга**.

> Технически: одностраничное веб-приложение на чистом ванильном стеке (без фреймворков и сборки),
> оптимизировано под iPhone Safari, деплой — статический (GitHub Pages).

## ✨ Основные механики
- **Драконы** — 15 видов, стадии роста, стихии, роли, пассивки, характеры, окрасы.
- **Яйца и инкубатор** — редкость, наследование окраса, пити-гарантия; яйца — ценная награда.
- **Бой** — пошаговая арена + аркадный бой в полёте; читаемые зоны, очередь, эффекты, ульты.
- **Исследование** — полёт над биомами, события, погода, боссы-владыки, % изучения.
- **Логово** — «дом стаи»: живые драконы, уровни логова, Активная стая + Заповедник.
- **Развитие** — Кузница, Скрещивание (Гнездилище), Шпиль талантов, Кодекс, вехи, ежедневки.
- **Прогрессия** — система «Следующий шаг»: всегда понятно, что делать и зачем.

## 🧱 Технологический стек
HTML + CSS (одна таблица) + 13 JS-модулей (общий глобальный скоуп, строгий порядок загрузки).
Отрисовка — DOM + Canvas + requestAnimationFrame. Звук — WebAudio-синтез. Хранение — localStorage.

## 📁 Структура проекта
```
index.html          — точка входа (подключает js/css/images из корня)
js/                 — 13 игровых модулей (00→11, строгий порядок)
css/                — стили (style.css)
images/             — графические ассеты (с fallback)
design/             — дизайн-система + токены (design/tokens/)
prompts/            — AI-библиотека промптов (категории + MASTER_*)
ai/                 — постоянный контекст и правила для ИИ
docs/               — аудиты, отчёты, история дизайна
tests/  tools/      — заготовки под тесты и инструменты
_archive/           — устаревшее/эксперименты (не в игре)
```
Подробно — [PROJECT_STRUCTURE.md](PROJECT_STRUCTURE.md).

## 🚀 Сборка и запуск
Сборка не требуется. Локально: открыть `index.html` через статический сервер (см. [INSTALL.md](INSTALL.md),
[BUILD.md](BUILD.md)). Деплой — GitHub Pages из корня (`.nojekyll`).

## 🛠 Процесс разработки
Правила — [CONTRIBUTING.md](CONTRIBUTING.md) и `ai/CODING_RULES.md`. Проверка: `node --check` каждого
модуля и конкатенации в порядке загрузки. При изменении ассетов — поднять `?v=N` в `index.html`.
