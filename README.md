# Драконис 3.0 · Кодекс Чешуи

Браузерная игра о разведении, развитии и приключениях драконов. Эта папка не требует сборки и готова для публикации через GitHub Pages.

## Локальный запуск

```powershell
python -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## Публикация на GitHub Pages

1. Создайте пустой репозиторий GitHub.
2. Загрузите содержимое этой папки в корень репозитория.
3. Откройте `Settings → Pages`.
4. Выберите `Deploy from a branch`, ветку `main`, каталог `/ (root)`.
5. Сохраните настройки и дождитесь публикации.

Файл `.nojekyll` уже включён.

## Структура

- `index.html` — точка входа;
- `css/` — стили и адаптивные макеты;
- `js/` — игровая логика;
- `images/` — используемые игровые изображения;
- `tests/` — синтаксическая и релизная проверка.

## Проверка

```powershell
powershell -ExecutionPolicy Bypass -File tests/run-tests.ps1
```
