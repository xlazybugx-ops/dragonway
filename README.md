# 🐉 Драконис · Кодекс Чешуи

Браузерная игра про коллекционирование, выращивание и развитие драконов. Сделана как один HTML-файл с папкой изображений — без установки, серверов и зависимостей. Достаточно открыть в браузере.

> Игра создана для ребёнка: добрая, без агрессии и давления, с акцентом на заботу о драконах и спокойное исследование.

## 🎮 Как играть

**В браузере по ссылке (GitHub Pages):**
👉 `https://ВАШ-ЛОГИН.github.io/НАЗВАНИЕ-РЕПОЗИТОРИЯ/`

*(ссылка заработает после включения GitHub Pages — см. ниже)*

**Локально на компьютере:**
Скачайте репозиторий и откройте `index.html` двойным щелчком. Для сохранения прогресса игру лучше открывать именно локально или через Pages — браузерное хранилище должно быть доступно.

## ✨ Что внутри

- **Коллекция драконов** — 15 видов, 5 стихий (огонь, лёд, яд, буря, тень), редкие окрасы.
- **Хаб «Драконьи Земли»** — живой главный экран с постройками-ссылками.
- **Забота** — драконов можно назвать, кормить и гладить.
- **Гнездо** — высиживание яиц с анимацией вылупления.
- **Странствия** — полёт-исследование по карте с точками интереса.
- **Турнир** — добрые поединки с мини-игрой «удар по таймингу».
- **Селекция** — генетика и выведение идеального дракона.
- **Шпиль Мироздания** — древо талантов до 100 уровня, своё у каждого вида.
- **Кузница** — реликвии и их улучшение.
- **Эволюция облика** — дракон меняет вид по мере роста уровня (стадии 1 / 25 / 60 / 100).

## 📁 Структура проекта

```
.
├── index.html        — сама игра (открыть в браузере)
├── images/           — изображения драконов
│   ├── ember_1.png   — {вид}_{стадия}.png
│   ├── ember_25.png
│   └── ...
├── README.md
└── LICENSE
```

**Имена файлов картинок:** `{вид}_{стадия}.png`, где стадия — `1`, `25`, `60` или `100` (по уровню дракона). Все 60 изображений (15 видов × 4 стадии) в комплекте. Если какого-то файла нет, игра сама нарисует дракона векторной графикой — ничего не сломается.

## 🚀 Включить GitHub Pages (играть по ссылке)

1. Загрузите все файлы в репозиторий на GitHub.
2. Откройте **Settings → Pages**.
3. В разделе **Build and deployment → Source** выберите **Deploy from a branch**.
4. Branch: **main**, папка: **/ (root)**. Нажмите **Save**.
5. Через минуту-две игра будет доступна по ссылке вида `https://ВАШ-ЛОГИН.github.io/НАЗВАНИЕ-РЕПОЗИТОРИЯ/`.

## 🛠 Технологии

Чистый HTML, CSS и JavaScript в одном файле. Графика — встроенный SVG и PNG-изображения. Никаких внешних библиотек и сборки.
