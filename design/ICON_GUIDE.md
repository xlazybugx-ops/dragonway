# ICON_GUIDE.md — единый язык иконок

Все иконки — одна семья: полуобъёмные, ручная роспись, мягкие блики, единая перспектива, единый
контур, единый источник света.

## Спецификация
- **Стиль:** semi-3D (полуобъём), нарисованы вручную, мягкая гуашь, скруглённые формы.
- **Перспектива:** лёгкий вид сверху-спереди (~15°), одинаковый для всех.
- **Свет:** один источник сверху-слева; мягкий блик сверху, лавандовая тень снизу-справа.
- **Контур:** единый ~2px, тёплый коричневый (`--ds-line-col`), слегка «от руки».
- **Материал:** матовый + один мягкий specular-блик; самоцветы/свечение — лёгкий глянец.
- **Сетка:** канвас 256×256, safe-area 12%, объект центрирован, единый визуальный вес.
- **Фон:** прозрачный (PNG/WebP с альфой) — иконка ложится на любую поверхность.
- **Цвет:** из палитры токенов; одна доминанта + акцент.

## Категории иконок
Ресурсы (золото, пыль, осколки, звёзды), яйца, стихии, награды/сундуки, статусы боя, навигация,
достижения, кнопки-действия, декор.

## Единообразие (чек-лист)
✓ один угол света ✓ одинаковый контур ✓ одинаковая перспектива ✓ одинаковый уровень блеска
✓ одинаковый визуальный вес ✓ прозрачный фон ✓ палитра токенов ✓ дружелюбная округлая форма.

## Запрещено
Плоский flat без объёма, разные углы света, разная толщина контура, реалистичные материалы, мелкие
детали, кислотные цвета, тени чёрным.

## Пример имени
`icon_gold.webp`, `icon_egg_legendary.webp`, `icon_status_shield.webp` (см. `ASSET_NAMING.md`).
