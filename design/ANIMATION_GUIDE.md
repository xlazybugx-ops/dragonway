# ANIMATION_GUIDE.md — оживление страниц книги

Приоритет — **процедурные CSS/Canvas-анимации**, а не тяжёлые видео/GIF. Всё масштабируемо,
переиспользуемо, оптимизировано для Safari, уважает `prefers-reduced-motion`.

## Каталог движений (как реализовывать)
| Движение | Техника | Заметки |
|---|---|---|
| Плавное покачивание объектов | CSS `transform: translateY/rotate` + `@keyframes`, ease-in-out | амплитуда малая, период 2–4с |
| Дыхание драконов | CSS `scale(1↔1.03)` по вертикали, 3–4с | или Canvas-скейл спрайта |
| Моргание | смена кадра/маски глаз раз в 3–6с | Canvas или два спрайта |
| Движение хвоста | Canvas-синус по кости / CSS rotate сегмента | лёгкое, живое |
| Колыхание травы | Canvas-синус по столбцам / CSS skew | параллакс-фон |
| Движение облаков | CSS translateX очень медленно (30–60с) | зациклено |
| Мерцание кристаллов | CSS opacity/shimmer-градиент | рассинхрон по фазе |
| Перелив свечения | CSS фон-позиция градиента | 2–3с |
| Сияние легендарных | конический градиент вращается + pulse box-shadow | из `--ds-glow-legend` |
| Параллакс фонов | слои с разной скоростью (scroll/gyro/pointer) | 3–4 слоя биома |
| Появления/исчезновения | CSS opacity+translateY, 180–360мс, `--ds-ease` | мягко |
| Микроанимации кнопок | press scale .97 + brightness, 160мс | уже есть в игре |
| Наведение/нажатие | hover lift / active scale | `--ds-t-fast` |
| Анимации карточек | tilt/scale на тап, flip на раскрытие | `--ds-t-mid` |
| Открытие наград | scale-pop + частицы + glow | Canvas+CSS |
| Частицы | Canvas-пул, тёплые точки | деградация на слабых |
| Переходы экранов | fade+slide 200–360мс | уже есть (`uxViewIn`) |

## Тайминги (токены)
fast 160 · mid 220 · slow 360 мс; единый ease `--ds-ease`. Микро — быстрые, атмосферные — медленные.

## Тайлинг живого логова
Драконы в логове: дыхание + моргание + хвост + редкие действия (сон/игра/полёт/у костра/наблюдение
дождя/уход за яйцами). Реализуется поверх существующего rAF-движка логова (поведение уже есть — задать
арт-параметры: амплитуды, периоды).

## Производительность (Safari)
- CSS-трансформы/opacity (GPU), без layout-thrash.
- Canvas: один слой, пул частиц, dt-clamp, самоостановка вне видимой области.
- Переиспользуемые узлы; никаких тяжёлых видео/GIF.
- `prefers-reduced-motion` отключает второстепенное движение.

## Чек-лист
✓ процедурно ✓ токенные тайминги ✓ GPU/Canvas-пул ✓ деградация ✓ reduced-motion ✓ переиспользуемо.
