# TYPOGRAPHY.md — типографика

Дружелюбная «книжная», не гравюрная. Токены — `tokens/`.

- **Дисплей:** `Baloo 2` / `Fredoka` (округлый, тёплый) — заголовки, кнопки.
- **Текст:** `Nunito` / `Quicksand` (мягкий) — основной текст.
- **Шкала:** hero `clamp(28px,7vw,44px)` · h2 `clamp(20px,5vw,28px)` · body 15px · caption 12px.
- **Цвет:** `--ds-ink` (#4A3B2E) на светлом; заголовки — тёплый коричневый/закат. Без чёрного-на-белом.
- **Начертания:** заголовки 600–700, текст 400–500. Межстрочный 1.4–1.5. Минимум текста, много воздуха.
- **Правила:** не более 2 гарнитур на экран; caps использовать редко (мелкие подписи).
- **Доступность:** режим «крупный текст» масштабирует шкалу; поддержать Dynamic Type в будущем.
