# CARD_GUIDE.md — карточки

- Фон `--ds-grad-card`, радиус `--ds-r-md`, тень `--ds-shadow-md`, inset-хайлайт `--ds-shadow-in`.
- Структура: иконка/арт (герой) → заголовок (display) → подпись (caption) → действие.
- Редкость/статус — ободковое свечение по токену редкости + бейдж/форма (не только цвет).
- Отклик: tilt/scale на тап (`--ds-t-mid`), flip на раскрытие деталей.
- Карточки коллекций/наград — единый размер в сетке, крупный радиус, мягкая тень.
- Не перегружать: главное на лице карточки, детали — по тапу.
