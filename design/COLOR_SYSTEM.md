# COLOR_SYSTEM.md — цветовая система

Правило №1: **никакой серости, чёрного, белого, кислоты и грязи.** Тёмное = тёплый коричневый,
светлое = кремовое. Все значения — в `tokens/tokens.{css,json}`.

## Базовая палитра
| Роль | Токен | HEX |
|---|---|---|
| Бумага (основа) | `--ds-paper` | `#FBF3E4` |
| Бумага 2 | `--ds-paper-2` | `#F5E7CE` |
| Чернила (текст) | `--ds-ink` | `#4A3B2E` |
| Чернила тихие | `--ds-ink-soft` | `#8A7B66` |

## Природа и атмосфера
| Роль | HEX |
|---|---|

| Бирюзовая вода | `#63D2C4` → `#3FB6AE` |
| Золотой солнечный свет | `#FFC96B` / сияние `#FFE3A3` |
| Оранжевый закат | `#FF9E5A` → `#F5794D` |
| Лавандовые тени | `#B7A8E0` → `#8E7CC7` |

| Небо (день/персик/лаванда) | `#CDE8F5` / `#FFD9B0` / `#D8CBF0` |

## Стихии (мягкие, добрые)
🔥 fire `#FF8A5B` · 🧊 frost `#7FD3E8` · 🍃 venom `#8FD46A` · 🌩 storm `#B79BEA` · 🌸 shade `#E08AB0`.
Тень — не мрак, а тёплая пыльно-розовая.

## Редкость (читается за <0.3с)
| Тир | HEX | Ощущение |
|---|---|---|
| Обычное | `#CBB89A` | тёплый камень |
| Необычное | `#7FC98A` | свежий лист |
| Редкое | `#6FB7E8` | ясное небо |
| Эпическое | `#B18AE0` | аметист |
| Легендарное | `#FFC24D` | золотое сияние |
| Древнее | `#F08AB4` | розово-магический глоу |

## Градиенты (готовые)
Небо, закат, лес, вода, золото, карточка — см. `--ds-grad-*`. Всегда мягкие вертикальные переходы.

## Правила применения
- Один доминирующий тёплый цвет на экран + 1–2 акцента. Остальное — приглушённая бумага.
- Насыщенность героя выше фона. Фон биома задаёт «температуру» сцены.
- Свечение/легендарность = золото/розовый глоу, не неон.
- Контраст текста: `--ds-ink` на бумаге (мягкий, не чёрный-на-белом).

## Доступность
Для режима «без опоры на цвет» редкость дублируется формой/узором/иконкой (см. `EGG_GUIDE`), а не
только цветом.
