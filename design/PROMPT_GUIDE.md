# PROMPT_GUIDE.md — как писать промпты для ассетов

1. Всегда: `MASTER_STYLE_PROMPT`-преамбула + блок категории (`prompts/PROMPT_TEMPLATES.md`) + NEGATIVE.
2. Палитра/свет/материалы — из токенов и `COLOR_SYSTEM.md`. Один сид-диапазон на серию.
3. Прозрачность и размер — по шаблону; итоговый файл — по габаритам заменяемого ассета (`ASSET_NAMING.md`).
4. Приёмка по чек-листу `STYLE_GUIDE.md` (тест «одной команды»). Брак → перегенерация.
5. Утверждённые ассеты — в референсы для image-to-image (консистентность серий).
Каталог мастер-промптов и категорий — `prompts/`. Индекс — `PROMPTS_INDEX.md`.
