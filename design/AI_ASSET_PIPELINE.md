# AI_ASSET_PIPELINE.md — конвейер ИИ-генерации ассетов

## Поток
1. **Бриф** → выбрать категорию (`prompts/<cat>`), мастер-промпт (`prompts/MASTER_*`).
2. **Промпт** = преамбула + блок + NEGATIVE + тех.параметры (размер/alpha/сид).
3. **Генерация** серией с фиксированной преамбулой; 2× + даунскейл.
4. **Приёмка** по `STYLE_GUIDE.md`; брак → перегенерация.
5. **Именование** строго по `ASSET_NAMING.md` (совпадение с ключами кода).
6. **Оптимизация**: WebP; фоны полёта ≤400–600 КБ; прозрачность где нужно.
7. **Интеграция**: положить в `images/` с тем же именем/габаритами → игра подхватит (есть fallback).
   Поднять `?v=N` в index.html.
8. **Проверка** на устройстве; занести в CHANGELOG.

## Гарантия единого стиля
Единый источник правды — `design/tokens/` + неизменяемая STYLE-преамбула. Любой ассет любой модели
проходит один и тот же чек-лист → «одна художественная команда».
