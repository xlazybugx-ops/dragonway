# FX_GUIDE.md — визуальные эффекты

Все эффекты — мягкие, пастельные, добрые. Приоритет — процедурные (CSS/Canvas), а не видео/GIF.
Масштабируемые и переиспользуемые. Оптимизированы для Safari (см. `ANIMATION_GUIDE`).

## Библиотека эффектов
| Эффект | Применение | Реализация |
|---|---|---|
| Мягкое свечение (glow) | редкость, награды, легендарки | CSS box-shadow/фильтр из `--ds-glow-*` |
| Искры/пыльца | вылупление, победа, epic+ яйца | Canvas-частицы (пул), мелкие тёплые точки |
| Блик/переливы (shimmer) | самоцветы, вода, кристаллы | CSS-градиент + анимация позиции |
| Ободковый ореол | легендарные/древние предметы | вращающийся конический градиент (CSS) |
| Пузырьки/споры/снежинки | частицы биомов | Canvas, редкие, медленные |
| Всплеск награды | открытие сундука/яйца | частицы + масштаб-поп карточки |
| Телеграф способности | читаемость боя | Canvas: мягкие круги/линии/цепочки (уже в игре) |
| Мягкая вспышка | левел-ап, крит | радиальный градиент, fade 0.6с (уже в игре) |

## Правила
- Цвета частиц — из палитры токенов (тёплые/пастель), прозрачность низкая, края мягкие.
- Плотность умеренная; на слабых устройствах — деградация (меньше частиц).
- Никаких резких вспышек, стробоскопа, кислотного неона, чёрного дыма.
- Все эффекты уважают `prefers-reduced-motion`.

## Производительность
- Частицы — через пул объектов на одном Canvas; фильтрация по времени жизни.
- Свечения — CSS (GPU), не JS-перерисовка.
- Переиспользуемые узлы (вспышка, церемония) — единичные, не плодятся.

## Чек-лист
✓ мягко/пастельно/добро ✓ токенные цвета ✓ процедурно (CSS/Canvas) ✓ переиспользуемо
✓ деградация на слабых ✓ reduced-motion ✓ без стробоскопа/неона.
