# GRID_SYSTEM.md — сетка и композиция

- **Базовая единица:** 4px. Все размеры/отступы кратны 4 (через spacing-токены).
- **Контентная ширина:** одна колонка, безопасные поля 16px; макс. читаемая ширина карточек ~520px.
- **Экран:** топбар (ресурсы) сверху · контент · нижняя навигация (таббар) снизу. Учитывать safe-area.
- **Композиция ассетов:** правило третей; один герой + ≤3 второстепенных; воздух вокруг.
- **Карточная сетка коллекций:** `auto-fill, minmax(82–110px, 1fr)`, gap 8–12px.
- **Слои сцен:** 3–4 плана для параллакса (фон/средний/передний/частицы).
