# UI_STYLE_GUIDE.md — интерфейс

Принцип: **современный iOS + тёплая книжка.** Крупные скругления, воздух, карточки, полупрозрачность,
мягкие тени. Никаких тяжёлых металлических RPG-панелей.

## Поверхности
- **Панель:** `--ds-grad-card`, радиус `--ds-r-lg` (28px), тень `--ds-shadow-md`, паддинг `--ds-gap-lg`.
- **Карточка:** та же логика, радиус `--ds-r-md`, лёгкий inset-хайлайт сверху (`--ds-shadow-in`).
- **Полупрозрачные слои (стекло):** `rgba(255,253,247,.72)` + `backdrop-filter: blur(8px)` для оверлеев/
  bottom sheet/таббара.
- Фон приложения — тёплая бумага/мягкое небо, не тёмный.

## Кнопки
- **Primary:** заливка `--ds-grad-gold`, текст `--ds-ink`, радиус `--ds-r-md`, тень `--ds-shadow-sm`,
  высота ≥48px. Press: scale .97 + brightness (160мс).
- **Soft/secondary:** бумага `--ds-paper-2`, контур 2px `--ds-line-col`.
- **Иконочная:** круглая (`--ds-r-pill`), 44×44 минимум.
- Без градиентного металла, без тяжёлых рамок.

## Bottom Sheet
- Снизу, радиус верхних углов `--ds-r-xl`, «ручка» сверху, фон-стекло, анимация подъёма 180–220мс,
  закрытие тапом по затемнённому (лавандовому, не чёрному) фону.

## Нижняя навигация
- Плавающая, стеклянная, 5 разделов, активный — тёплый акцент + мягкое поднятие иконки, safe-area.

## Карточки наград/коллекции
- Крупный радиус, мягкая тень, ободковое свечение по редкости (`--ds-glow-*`), лёгкий tilt на нажатии.

## Типографика
- Заголовки — `--ds-font-display` (округлый), текст — `--ds-font-body`. Минимум текста, много воздуха.
- Цвет текста `--ds-ink` на светлом. Заголовки могут быть тёплого коричневого или закатного.

## Иерархия
Один герой на экран, ≤3 второстепенных. Второстепенное — тише по насыщенности и мельче. Служебное —
под спойлер/лист.

## Запрещено
Тёмные металлические панели, острые углы, чёрные тени, плотные таблицы, мелкий засечный текст, кислота.

## Пример компонента (на токенах)
```css
.ds-card{background:var(--ds-grad-card);border-radius:var(--ds-r-lg);box-shadow:var(--ds-shadow-md);
  padding:var(--ds-gap-lg);color:var(--ds-ink);font-family:var(--ds-font-body)}
.ds-btn{background:var(--ds-grad-gold);border:none;border-radius:var(--ds-r-md);min-height:48px;
  font-family:var(--ds-font-display);color:var(--ds-ink);box-shadow:var(--ds-shadow-sm);
  transition:transform var(--ds-t-fast) var(--ds-ease),filter var(--ds-t-fast)}
.ds-btn:active{transform:scale(.97);filter:brightness(1.08)}
```
