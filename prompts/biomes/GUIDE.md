# BIOME_GUIDE.md — биомы (каждый — своя сказка)

Каждый биом — отдельная страница книги: собственная палитра, растительность, атмосфера, свет, частицы,
цвет тумана и теней. При этом все — одна семья (Cozy Storybook).

## Общие правила
- Фон тише героя: ниже контраст, мягкий градиент неба сверху.
- Многослойность для параллакса (см. `ANIMATION_GUIDE`): дальние горы → средний план → передний.
- Свет тёплый, направленный; тени биома тонируются в его «теневой» цвет.

## Элементы природы (единый стиль)
Растения, деревья, камни, горы, реки, водопады — мягкие округлые формы, ручная роспись, свет
сверху-слева. Вода — бирюзовая с мягкими бликами; светящиеся растения — акцентные глоу-точки.

## Ярусы полёта (tier 1/2/3)
Файлы `fly_{ключ}_{tier}.webp`. С глубиной (tier↑) — атмосфера насыщеннее, больше волшебных частиц,
но палитра биома сохраняется. Есть процедурный fallback (палитры в коде) — новые фоны опциональны.

## Запрещено
Grimdark, реалистичные текстуры, серый туман, чёрные тени, кислотная зелень/синь, страшная атмосфера.

## Чек-лист биома
✓ уникальная палитра-настроение ✓ единый стиль росписи ✓ слои для параллакса ✓ тёплый свет
✓ фирменные частицы/туман/тени ✓ дружелюбие ✓ фон тише героя.
