# DRAGON_GUIDE.md — драконы-персонажи

Каждый дракон — **персонаж, а не монстр.** Большие выразительные глаза, понятная эмоция, добрая улыбка,
уникальный узнаваемый силуэт (читается даже без цвета).

## Характер-дизайн
- **Глаза:** крупные, блестящие, добрые; заметный catchlight (блик). Главный носитель эмоции.
- **Лицо:** мягкое, приветливое; лёгкая улыбка по умолчанию. Никаких клыков-угроз, злого прищура.
- **Пропорции:** «пухлые», голова относительно крупная (chibi-дружелюбие), мягкий живот, короткие
  лапки у детёнышей → стройнее у взрослых стадий.
- **Силуэт:** уникальный и читаемый в чёрном контуре — форма крыльев/рогов/хвоста узнаётся сразу.

## Стадии роста (совпадают с игрой)
Файлы: `{id}_1`, `{id}_25`, `{id}_60`, `{id}_100` (пороги `stageForLevel`). Стадии — один и тот же
персонаж, взрослеющий: детёныш (крупная голова, большие глаза) → подросток → взрослый → величавый
(ур.100, лёгкое сияние, богаче силуэт). Сохранять узнаваемость между стадиями.

## Стихии → настроение палитры (мягкие)
🔥 fire — тёплый оранж, уютный; 🧊 frost — нежно-голубой, спокойный; 🍃 venom — свежий зелёный, живой;
🌩 storm — мягкая сирень, игривый; 🌸 shade — пыльно-розовый, загадочный-но-добрый (НЕ страшный).

## 15 видов (id проекта — сохранять имена файлов)
ember, cinderpup, magma, pyrelord (огонь) · glacier, permafrost, aurora (лёд) · sporewing, blightfang,
worldserpent (яд) · tempest, thundercall (буря) · umbra, nightwyrm, voidmaw (тень).
Даже «voidmaw»/«nightwyrm» — добрые и обаятельные (загадочные, не жуткие).

## Свет и подача
Свет сверху-слева, тёплый rim по контуру, мягкая контактная тень (лавандовая). Поза — расслабленная,
приветливая; лёгкий разворот 3/4. Фон в спрайтах — прозрачный.

## Эмоции (для микроанимаций/поз)
радость, любопытство, сон, игра, удивление — базовый набор выражений (см. `ANIMATION_GUIDE` для
дыхания/моргания/хвоста).

## Запрещено
Страшные/агрессивные морды, реализм чешуи, тёмные грязные тона, острые «боевые» силуэты, anime-глаза,
photoreal, low-poly.

## Чек-лист приёмки дракона
✓ большие добрые глаза ✓ улыбка/понятная эмоция ✓ уникальный силуэт без цвета ✓ мягкие формы
✓ палитра стихии из токенов ✓ свет сверху-слева ✓ прозрачный фон ✓ узнаваемость между стадиями.
