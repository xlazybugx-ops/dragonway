# INSTALL

## Игроку
Открой ссылку на GitHub Pages в Safari на iPhone → «Поделиться» → «На экран Домой» для полноэкранного
режима. Прогресс сохраняется локально; есть экспорт/импорт сейва в разделе Профиль.

## Разработчику
1. Клонируй репозиторий.
2. Запусти статический сервер из корня (см. `BUILD.md`).
3. Открой в браузере; для мобильной проверки — DevTools device mode или реальный iPhone.

## Деплой
GitHub Pages из корня ветки (наличие `.nojekyll` обязательно). Никаких шагов сборки.
